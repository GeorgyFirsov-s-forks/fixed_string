1. Replaces the contents with a copy of `other`.
2. Replaces the contents with a copy of `array`.

## Parameters

* `other` - another fixed string to use as source to initialize the string with
* `array` - reference to an array of characters to use as source to initialize the fixed string with

## Complexity

1-2. Linear in `N`
